# Movie Playlist Manager

A web application where users can search for movies and add their favorite ones to a personal playlist based on the ratings they give. The platform allows users to organize and manage their movie collection easily.

## Features

- 🔍 Search for movies
- ⭐ Add movies to a personal playlist with custom ratings
- ✏️ Edit movie details and ratings
- ❌ Remove movies from the playlist
- 📋 Maintain a personalized movie collection

## Functionality

Users can:
- Browse and search for movies
- Add movies they like to their personal playlist
- Assign a rating to each movie
- Update or modify movie information
- Delete movies from their playlist whenever they want

## Purpose

This project demonstrates basic CRUD (Create, Read, Update, Delete) operations and user interaction with a movie database in a simple and organized interface.

# Project Dependencies

This project includes a requirements.txt file containing all the necessary Python packages.

Install them using:

pip install -r requirements.txt

This command will automatically install all required libraries listed in the file.
